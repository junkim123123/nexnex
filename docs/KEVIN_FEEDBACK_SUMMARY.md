# Kevin Park의 NexSupply 피드백 요약

## 🎯 핵심 메시지

> **"이거… 진짜로 내 인생 바꿔놓을 사이트임"**

Kevin (24, LA, 초보 아마존 FBA 셀러)이 NexSupply를 사용한 후 실제로 느낀 감정과 행동 변화를 기록한 문서입니다.

---

## ✅ Kevin이 좋아한 것들

### 1. Simple Mode (Beginner Protection Mode)
- **"복잡한 숫자 다 필요 없고, 딱 '이거 해? 말아?'만 알려주는 거"**
- 최종 이익만 크게 표시
- 3개 행동 버튼만 제공
- Advanced 모드로 전환 가능

### 2. Hidden Costs Breakdown
- **"아 이제 알겠다... 나 지금까지 엑셀에서 '판매가 - 원가'만 계산했는데"**
- FBA Referral Fee, Fulfillment, Return Processing, Storage, Ads Cost 모두 분리
- "왜 내가 월 25K 매출인데 이익이 0이었구나" 깨달음

### 3. Verdict Go/No-Go
- **"1초 만에 '이거 망한 거네'라고 인지"**
- 빨간색/초록색으로 즉각적 판단
- 구체적인 이유와 다음 액션 제시

### 4. MOQ Negotiation 도우미
- **"오 이거 진짜 쓸 만한데?"**
- 전략적 협상 메시지 템플릿 제공
- 성공률 70% 전략 제시

### 5. Channel Comparison
- **"'아… 내가 왜 아마존만 고집했지?' 진짜 깨달음 옴"**
- FBA vs DTC vs Wholesale 한눈에 비교
- 각 채널의 실제 마진 표시

---

## ❌ Kevin이 아쉬워한 것들

### 1. 입력 화면
- **"어디까지 써야 하지"** 불안감
- 템플릿 버튼 선택 여부가 불명확
- Advanced options가 어려워 보임

**→ 해결:** "딱 이 네 가지만 써도 됩니다" 명확한 안내 추가

### 2. 결과 화면 정보 과부하
- **"12개 섹션 다 봐야 하는 거? 근데 난 그냥 최종 이익만 알고 싶은데"**
- 처음에는 너무 많은 정보

**→ 해결:** Simple Mode 추가 (최종 이익 + 3개 액션만)

### 3. 모바일 사용성
- **"지하철에서 '이거 $2.30 남는다'는 건 1초에 알고 싶은데, 3개 탭 열어야 함"**
- 테이블 가로 스크롤이 부드럽지 않음

**→ 해결:** 모바일 친화적 카드 레이아웃 및 부드러운 스크롤 추가
