# NexSupply 앱 - 100명 내부 직원 피드백 시뮬레이션

이 문서는 다양한 페르소나의 피드백을 시뮬레이션하고 개선 작업을 추적합니다.

## 페르소나 분류

### 1. 초보자 그룹 (30명)
- **Kevin (초보 수입업자)**: 처음 수입하는 사람, 간단한 UI 필요
- **Sarah (소상공인)**: 작은 규모, 명확한 설명 필요
- **Mike (부업 시작)**: 부업으로 시작, 빠른 결정 필요

### 2. 경험자 그룹 (25명)
- **Mia (경험 있는 구매자)**: 상세한 데이터, 비교 기능 필요
- **David (전문 수입업자)**: 고급 옵션, 커스터마이징 필요
- **Lisa (제품 매니저)**: 경쟁 분석, 시장 인사이트 필요

### 3. 재무/운영 그룹 (20명)
- **CFO (재무 담당자)**: ROI, 현금흐름, 재무 지표
- **Operations Manager**: 리드타임, 재고 관리, 공급망
- **Procurement**: 협상 전략, 공급업체 비교

### 4. 기술/개발 그룹 (15명)
- **Developer**: 디버그 모드, API 접근, 데이터 구조
- **Data Analyst**: 통계, 트렌드, 데이터 시각화
- **QA Engineer**: 버그, 엣지 케이스, 사용성 테스트

### 5. 마케팅/고객 서비스 (10명)
- **Marketing Manager**: 브랜딩, 고객 경험, 메시징
- **Customer Service**: 사용자 친화성, 에러 메시지, 도움말
